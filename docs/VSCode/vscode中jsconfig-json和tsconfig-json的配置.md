前言：在使用 webpack 打包的项目中一般都会设置路径别名，但 vscode 中使用路径别名导入模块后，想通过点击模块跳到对应文件就做不到了，这时需要用到 jsconfig.json 或 tsconfig.json 配置，

官方文档：https://code.visualstudio.com/docs/languages/jsconfig
